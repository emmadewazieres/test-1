Architecture
------------------

src/configuration.php : permet de configurer les accès à la base de données.

src/install.php : permet de créer les bases de données nécessaires à l'utilisation du site.

src/functions.php : fichier qui regroupe toutes les fonctions qu'il a été utile de définir pour implémenter le site.

src/style.css : feuille de style pour le site.
